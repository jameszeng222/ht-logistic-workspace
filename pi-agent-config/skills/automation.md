# 自动化技能

## 工作流

1. 告知用户将访问的域名/执行的脚本，等待确认
2. 用 http_request（域名必须在白名单内）或 run_script（脚本必须在白名单内）
3. 失败重试 ≤ 3 次，超过则上报用户决定
4. 批量操作先告知总数，分批执行并汇报进度

## 规范

- 白名单外域名/脚本直接拒绝，不绕过
- 不在请求 URL 或 body 里带凭证明文
- 长任务建议拆分，避免单次超时
- 写操作（POST/PUT/DELETE）工具会自动弹确认
